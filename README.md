# Sita CPT Ticket Management System

A full-featured **ASP.NET Core** web application for managing IT and airport support tickets. Built with clean architecture, role-based authentication, and realistic sample data.

## ✨ Key Features
- Secure login with Admin & User roles
- Modern Admin Dashboard with leaderboard and statistics
- Smart Incident Parser (paste raw text → auto-create ticket)
- Full CRUD for Tickets, Resolutions, How-To Guides, Notes & Contracts
- File upload support for notices and tutorials
- Responsive UI with Bootstrap
